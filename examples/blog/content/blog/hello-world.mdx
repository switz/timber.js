# Hello World

Welcome to my blog built with **timber.js** and content collections.

This post demonstrates:

- MDX rendering as React Server Components
- Frontmatter extraction and type-safe schemas
- Content collections with Zod validation

## Why timber.js?

timber.js is a Vite-native React framework for Cloudflare Workers. It prioritizes correct HTTP semantics, real status codes, and pages that work without JavaScript.

## Getting Started

Install the package:

```bash
pnpm add @timber-js/app
```

That's it — you're ready to build.
