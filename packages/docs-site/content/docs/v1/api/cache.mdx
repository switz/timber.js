# @timber/app/cache

Caching primitives and cache handler interface. Import from `@timber/app/cache`.

## `createCache(fn, options)`

Wrap an async function with cross-request caching.

```ts
import { createCache } from '@timber/app/cache';

const getUser = createCache(
  async (userId: string) => {
    return db.users.findUnique({ where: { id: userId } });
  },
  {
    ttl: 60,
    key: (userId) => `user:${userId}`,
    staleWhileRevalidate: true,
    tags: (userId) => [`user:${userId}`],
  }
);
```

### `CacheOptions<Fn>`

| Option | Type | Description |
|--------|------|-------------|
| `ttl` | `number` | Time-to-live in seconds |
| `key` | `(...args) => string` | Custom cache key function (optional) |
| `staleWhileRevalidate` | `boolean` | Serve stale while refetching in background |
| `tags` | `string[] \| (...args) => string[]` | Tags for invalidation |

Default key: SHA-256 hash of function identity + deterministic JSON serialization of arguments.

## `CacheHandler` Interface

Implement this to plug in a custom cache backend.

```ts
interface CacheHandler {
  get(key: string): Promise<{ value: unknown; tags: string[] } | null>;
  set(key: string, value: unknown, options: { ttl: number; tags: string[] }): Promise<void>;
  invalidate(options: { tag: string }): Promise<void>;
}
```

## `MemoryCacheHandler`

In-memory cache with LRU eviction. The default handler.

```ts
import { MemoryCacheHandler } from '@timber/app/cache';

const handler = new MemoryCacheHandler({ maxSize: 1000 });
handler.size; // Current number of entries
```

### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `maxSize` | `number` | `1000` | Maximum entries before LRU eviction |

## `RedisCacheHandler`

Redis-backed cache handler.

```ts
import { RedisCacheHandler } from '@timber/app/cache';
import { createClient } from 'redis';

const redis = createClient({ url: 'redis://localhost:6379' });
const handler = new RedisCacheHandler(redis);
```

## Invalidation

### `revalidateTag(tag)`

Invalidate all cache entries with the given tag. Re-exported from `@timber/app/server` for convenience.

```ts
import { revalidateTag } from '@timber/app/cache';

revalidateTag('products');
revalidateTag(`user:${userId}`);
```

## Singleflight

### `createSingleflight()`

Request coalescing — when multiple concurrent calls miss the cache for the same key, only one executes the underlying function. Built into `createCache()` automatically.

```ts
import { createSingleflight } from '@timber/app/cache';

const sf = createSingleflight();
const result = await sf.do('key', () => expensiveOperation());
```

## Utilities

### `stableStringify(value)`

Deterministic JSON serialization with sorted object keys. Used internally for cache key generation.
