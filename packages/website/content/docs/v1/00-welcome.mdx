# Welcome to timber.js

timber.js is a React web framework built on Vite. It runs natively on Cloudflare Workers, Node.js, Bun, and many other platforms. It has fast build times, fast incremental compilation, and a low resource footprint.

It provides an emphasis on clarity, developer experience, and web standards. It's designed to be deployed on server and serverless platforms, but it can generate a static site and even deploy websites with _0kb of client-javascript_.

It takes inspiration from next.js and a long history of web frameworks, but redirects intentions towards a different set of values. One that understands the inherent value of both the server _and_ the client, but does not attempt to hide that complexity with _magic_.

With Timber you get:

- Proper Status Code Handling
- Graceful Degradation w/ Javascript Disabled
- Real Middleware
- Fully typed and integrated schemas that work across the server and the client, for:
  - search params
  - url params
  - cookies
  - server actions
  - forms

You will understand how and why your websites work with less confusion. Your sites will gracefully degrade without javascript. And we'll all experience a more fundamentally calm web.

Please give it a try, and lend yourself some patience to understand its concepts. Sharpen your axe before cutting down the tree.
